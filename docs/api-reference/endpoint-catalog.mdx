# API Endpoint Catalog

The OpenRouter API provides **49 endpoints** across **23 categories**.

## Public Endpoints (No Authentication)

These endpoints can be accessed without an API key:

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/models` | List all available models |
| GET | `/models/{author}/{slug}/endpoints` | List endpoints for a specific model |

## Authentication Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/auth/keys` | Exchange authorization code for API key |
| POST | `/auth/keys/code` | Create authorization code |
| GET | `/key` | Get current API key metadata |
| GET | `/keys` | List API keys |
| POST | `/keys` | Create a new API key |
| GET | `/keys/{hash}` | Get a single API key |
| PATCH | `/keys/{hash}` | Update an API key |
| DELETE | `/keys/{hash}` | Delete an API key |
