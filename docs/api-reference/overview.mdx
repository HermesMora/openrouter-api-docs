# API Overview

The OpenRouter API provides a unified interface to hundreds of AI models.

## Base URL

```
https://openrouter.ai/api/v1
```

## Authentication

All requests require a Bearer token in the `Authorization` header:

```bash
Authorization: Bearer *** Versioning

The API is versioned via URL path. Current version: `v1`.

## Content Types

- Request: `application/json`
- Response: `application/json` or `text/event-stream` (for streaming)

## Rate Limiting

Rate limits are returned in response headers:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Maximum requests allowed |
| `X-RateLimit-Remaining` | Requests remaining in window |
| `X-RateLimit-Reset` | Unix timestamp when limit resets |
